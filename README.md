# `Water-the-soil app`

React + Redux example app 

## `Language`
-TypeScript

## `Stuck technologies`

-React

-Redux

### `First start`

1. Install packages using commands npm i
2. Run npm start to start app
3. Open http://localhost:3000 to view app in the browser
